# CRUD VueJS
Aprendendo VueJS, video aulas do canal Descompila no Youtube.

# Install NodeJs
- https://nodejs.org/

# Back-end - Na pasta backend


## Instalar dependências
```
npm install
```

### Subir o servidor
```
npm start
```

# Front-end - Na pasta frontend

## Instalar dependências
```
npm install
```

### Rodar a aplicação
```
npm run serve
```
